---
description: Добавить запись о текущем проходе в docs/DEVLOG.md
allowed-tools: Read, Edit, Bash(git log *), Bash(git branch *), Bash(git status *)
---

Добавь запись в начало журнала `docs/DEVLOG.md` (сразу после заголовка `# DEVLOG`, новые записи сверху) в формате:

```
## ГГГГ-ММ-ДД · <ветка> · <короткий hash последнего коммита или «не закоммичено»>

- **Что сделано:** 2–5 пунктов, по фактам этой сессии (не пересказывай план — только сделанное).
- **Дальше:** 1–3 пункта следующих шагов; задачи, которые не берём сейчас, — перенеси в BACKLOG.md.
```

Дату возьми текущую, ветку и hash — из `git branch --show-current` и `git log -1 --format=%h`. Содержимое пунктов — из контекста текущего разговора: что реально было сделано в этом проходе. Если пользователь передал аргументы ($ARGUMENTS) — используй их как основу «Что сделано»/«Дальше».
