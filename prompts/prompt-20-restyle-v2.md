## Scope

**Делаем:**
1. **Токены** (`frontend/src/index.css`): палитра v2 (фон `#0b0d13`, поверхность `rgba(24,26,34,.92)`, чернила `#e7e8ee` + ступени, идентичность: фиолет/циан/оранж/синий; светофор статусов сохранить), радиусы (карточка 14px / бейдж 10px / пилюля 99px). Старые токены переназначить, а не плодить дубли; хардкод-долг D2 по возможности втянуть в токены (фиксировать в «Открытых вопросах», если что-то осталось).
2. **`CommandPanel` в шкуре v2** (`os.css` + разметка, логика нетронута):
   - полотно: фон + сетка 34px; плавающие пилюли (лого/крошка, поиск, «ход», аватар) вместо сплошных баров; rail блоков фирмы — компактная плавающая колонка слева (те же демо-данные);
   - узлы карты (реальные и демо): карточки v2 — иконка-бейдж с цветом ветки, имя, владелец, чип «N KPI»/«туман»; туман — пунктир + приглушение; выделенный — фиолетовая обводка со свечением; рёбра — пунктирные, цвет ветки, бегущие (`os-flow` сохранить);
   - **вертикаль собеседников** у правого края: орбы советников (демо `ADVISOR_SLOTS`) + бейджи непрочитанного (демо-числа) + пунктирный «+ слот»; «шёпот» — последняя реплика активного советника одной строкой у орба;
   - **оверлей разговора**: клик по орбу → карта в `blur(3px) brightness(.55)`, большой орб (цвет агента), колонка тем (демо: 3 темы с привязкой к целям), переписка (существующие демо-чаты `slot.chat` + локальный ввод, как сейчас), ввод-пилюля с иконкой микрофона (иконка без функции); закрытие — клик по фону/✕; `Escape` тоже закрывает;
   - **компонент `AdvisorOrb`** (`frontend/src/os/AdvisorOrb.tsx`): canvas, точки по сфере Фибоначчи, дыхание+вращение+шум (портировать алгоритм из эталонного HTML), параметры — размер и hue; при `prefers-reduced-motion: reduce` — статичный кадр без rAF-цикла.
3. **`RealGoalCard` и `GoalCard`** — перекрасить токенами/классами v2 (радиусы, поверхности, пилюли), структуру и логику не менять.
4. **Часть II `Visual_Reference.md`** — обновить под новый as-built: токены D2, геометрия D4, новые компоненты в D5 (орб, вертикаль, оверлей, шёпот), движение D7 (дыхание орба + правило reduced-motion), скрин нового состояния в D10 (`os-panel-v2-asbuilt.png`).

**НЕ делаем (сознательно):**
- Логику, состояния, API-клиент, backend — не трогать. Никаких новых данных: советники, темы, бейджи — демо.
- Карточку агента (настройка орба «как персонажа») — НЕ в этом срезе; клик по большому орбу ничего не делает.
- Туман-рой, «галактику», 2.5D-параллакс — не реализовывать (направления).
- Голос — только иконка микрофона.
- Связи/циклы KPI на карте не визуализировать (ADR-0004, открыт).
- Новых цветов/радиусов сверх Части III не вводить.

## Ограничения
- UI-текст — русский. PowerShell 5.1 — команды по одной, без `&&`; в venv только `python -m X`.
- Доступность: фокус-обводки на всех интерактивных элементах сохранить; орб-кнопки — `role="button"` + `aria-label`; reduced-motion гасит ВСЁ движение, включая canvas.
- После merge в `main` — `git push origin main`.
