# prompts/

Промпты для Claude Code. Пишет их Cowork (см. `COWORK.md` в корне), запускает — владелец.

## Правила

- **Активные** (написаны, ещё не выполнены) — лежат в корне `prompts/`.
- **Выполненные и смёрженные** — переезжают в `prompts/_done/`.
- Именование: `prompt-NN-краткое-имя.md`, NN — порядковый номер.
- `_done/` — не свалка, а **образец стиля**: смотри туда, если сомневаешься в формате промпта.

## Активные

## Что дальше по плану

- **Шаг 3c** — детект циклов в графе связей + подсветка + «судья» (ближайший общий структурный предок узлов цикла) + ручной флаг «осознанная ставка».
- **Шаг 3b** — составные KPI: качественная цель из взвешенных факторов-KPI (гудвилл = 0.3·отзывы + 0.3·LTV + 0.4·брак). Расширяет правило тумана: «есть таргет ИЛИ есть формула».
- Далее — ресурсные блоки (Финансы/Персонал/Продажи/Товары/Производство) как предпосылка «факта» KPI (ADR-0005).
